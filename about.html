Created by Henry James

The synth is powered by Tone.js and the visuals powered by YouTube's iframe API and some JavaScript to fade the videos

Below is a mapping of the weather data to the parts of the piece

Weather data Weather Drone Description
Current cloudiness 0-100% and daily cloudiness Max opacity Cloudiness controls how vibrant the videos are
Current wind speed Auto filter LFO speed The faster the current wind the faster the filters oscillation
Current rin in mm Pink noise volume The more rain per mm the louder the pink noise becomes
